Kursk Meetup — это сообщество программистов, дизайнеров, бизнесменов, студентов и гиков Курска.

На встречах делимся знаниями, раздаем полезные советы, рассказываем о последних работах, обсуждаем взаимоотношения с клиентами и профессиональное развитие.

Встречаемся раз в две недели по субботам в 11:00 [в кафе Museberry](https://www.google.ru/maps/place/Museberry/@51.741819,36.1925466,3a,75y,159.2h,88.88t/data=!3m6!1e1!3m4!1sDKZuGOSxnS1B_zQ2hHOOyQ!2e0!7i13312!8i6656!4m2!3m1!1s0x0:0x54c933bca61e5d5e!6m1!1e1).

Видеозаписи встреч выкладываем [в Ютюбе](http://www.youtube.com/channel/UCTyKgAbpgMEwfETFEE4aFpw). Пишем о новостях сообщества и вакансиях [в группе Вконтакте](http://vk.com/kurskmeetup) и [Слаке](http://kurskmeetup-slackin.herokuapp.com/).

Дата ближайшей встречи еще не назначена.
